# Inline scripts and interactive elements leak into extracted content

## Context

Bun.com embeds clipboard-copy JavaScript handlers as inline content within code blocks. The first code block in the article had a `document.addEventListener("DOMContentLoaded", ...)` clipboard handler concatenated directly into the terminal output, producing a 500-character garbage line.

## Gap

The HTML has:

```html
<pre><code class="language-text">❯ bun './**/*.html' document.addEventListener("DOMContentLoaded",()=>{const clicker=...}); Bun v 1.3.12 ready in 6.62 ms → http://localhost:3000/...</code></pre>
```

The clipboard handler JS was literally part of the `<code>` text content — likely injected by the site's build system for the "copy to clipboard" feature. h2md extracted it verbatim.

Additionally, `<div class="CopyIcon">` button elements appeared inside code block containers and were extracted as text.

## Responsibility

This is a content-extraction quality issue. The tool should filter out known non-content patterns from within code blocks. The agent had to manually identify and surgically remove the injected JS, reconstruct the intended terminal output, and format it properly — a brittle, error-prone process.

## Suggestion

1. **Strip copy-button elements**: Remove elements matching common copy-button patterns (`class*="copy"`, `class*="clipboard"`, copy icon SVGs) from code block containers before text extraction.

2. **Detect injected clipboard scripts**: Look for `addEventListener`, `clipboard.writeText`, `navigator.clipboard` patterns within `<code>` text content. When found at a boundary between real code/output and script code, split and discard the script portion.

3. **General heuristic**: If a `<code>` block contains `document.querySelector` or `addEventListener` and the surrounding context has no `<script>` tag, flag it in notes.md with the exact boundary for easy manual removal.

## Impact

Prevents garbage output in the most visible part of the article (the first code block). Small code change, high-visibility improvement.
